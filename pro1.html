Smart Submersible Pump Controller

Project Description

This project controls submersible motor using ESP32 and SIM 800L to operate remotely from anywhere at any time.It prevents dry run and saves Pump life.To reduce man power to operate. To controll by "SMS" via mobile phone.

Components Used

  • ESP32 Microcontroller
  • GSM Module
  • Relay Module
  • Current Sensor
  • Voltage Sensor
  • Temperature Sensor
  • LCD Display
